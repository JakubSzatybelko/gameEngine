# Minimal Game Engine (vibecoded) - code not reviewed yet

A game engine project for building simple 2D games. used to prototype game ideas quickly and efficiently.

## Getting Started

1. Clone the repository
2. Install dependencies
3. Build and run
4. (optional) delete example code.

## Features

- Rendering system
- Input handling
- Asset loading
- Scene management
- Basic physics and collision detection
- Audio support
